

Via Laravel Installer: composer global require laravel/installer
Via Composer Create-Project: composer create-project --prefer-dist laravel/laravel project_name
Start project: php artisan serve
migrate database: php artisan migrate
delete all table and migrate: php artisan migrate:refresh
Check laravel Framework Version: php artisan --version
create table: php artisan make:migration create_tableName_table --create=tableName
create controller and model: php artisan make:controller controllerName --resource --model=modelName
db table with migration: php artisan make:migration create_orders_table

no delete, no chage just do it: php artisan migrate:fresh
